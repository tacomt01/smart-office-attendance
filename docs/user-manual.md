## 1. 🔑 เข้าสู่ระบบ
1. กรอก **Email** และ **Password**
2. กด **"เข้าสู่ระบบ"** → ขึ้น ✓ สีเขียว แล้วเข้าหน้า Dashboard
3. ออกจากระบบ: ไอคอนโปรไฟล์ (มุมขวาบน) → **ออกจากระบบ**

---

## 2. ☁️ อัปโหลดไฟล์ (Admin)
1. เมนู **อัปโหลด** → ลากไฟล์มาวาง หรือคลิกเลือกไฟล์
2. รองรับ **Excel (.xls, .xlsx) หรือ CSV** รูปแบบ Daily Attendance Matrix แนวนอนเท่านั้น
3. กด **"อัปโหลดและประมวลผล"** → ดูสรุป: จำนวนพนักงาน, records, ช่วงวันที่

> **รูปแบบไฟล์:** แถว 1-2 = meta (ข้าม), แถว 3-4 = header (วันที่ MM/DD), แถว 5+ = ชื่อพนักงาน + ค่า `เวลาเข้า-เวลาออก`
> ไฟล์ที่ **Export** ออกจากระบบนำกลับมาอัปโหลดซ้ำได้ทันที (round-trip)

## 4. 🤖 AI Chat — ถาม-วิเคราะห์ + ส่งออก

### การสนทนา
- พิมพ์คำถามภาษาไทย เช่น *"ใครมาสายบ่อยที่สุด?"*, *"เลือก 3 คนที่มาทำงานดีที่สุด แสดงเป็นตาราง"*
- AI ตอบโดยอ้างอิงข้อมูลจริง และ **จำบริบทการคุยก่อนหน้า** (ถามต่อเนื่องได้ เช่น "แล้วคนที่ 2 ล่ะ")

### สิทธิ์ตาม role
- **Admin:** ถาม/วิเคราะห์ได้ทั้งบริษัท + สั่ง export ได้
- **Employee:** ถามได้เฉพาะข้อมูลการเข้างาน **ของตัวเอง** เท่านั้น — ถ้าถามถึงพนักงานคนอื่นระบบจะตอบว่า *"ไม่มีสิทธิ์เข้าถึง..."* และ **สั่ง export ไม่ได้**

### เลือก AI / Model
- แถบบนขวา: ปุ่ม **AI** (Claude / Gemini / Groq) + dropdown **Model** ของ provider นั้น — **ค่าเริ่มต้น = Gemini**
- ค่าที่เลือกถูกจำไว้อัตโนมัติ
> 🛟 เลือก **Claude** ได้ แต่บน Windows ภาษาไทยอาจเพี้ยน ระบบจึง **fallback ไป Gemini ให้อัตโนมัติ** เพื่อให้ได้คำตอบที่ถูกต้องเสมอ

| Provider | Models |
| :--- | :--- |
| Claude (local) | Haiku 4.5 · Sonnet 4.6 · Opus 4.8 |
| Gemini | Gemini 3.5 Flash · 2.5 Pro · 2.5 Flash |
| Groq | Llama 3.3 70B · Llama 3.1 8B · Llama 4 Scout |

### ประวัติแชท (แยกตาม user)
- แถบซ้าย: **แชทใหม่** / **ค้นหา** / **ล้างประวัติ** — เก็บหลาย session ในเครื่อง 24 ชม.
- ประวัติของแต่ละบัญชี **แยกจากกัน** (login คนละ user บนเครื่องเดียวกัน จะไม่เห็นประวัติของอีกคน)

### 🆕 ส่งออก Excel แบบกรองด้วยคำสั่ง (Admin)
พิมพ์คำสั่งที่มีคำว่า **export / ส่งออก / ดาวน์โหลด** พร้อมเงื่อนไข ระบบจะดึงตัวกรองให้อัตโนมัติ แล้วขึ้นปุ่ม **"ดาวน์โหลด Excel"**

| ตัวอย่างคำสั่ง | ผลลัพธ์ที่กรอง |
| :--- | :--- |
| `export ทั้งหมด` | ทุกคน ทุกวัน |
| `export ของ สมชาย` | เฉพาะพนักงานชื่อสมชาย |
| `ส่งออก เดือนมิถุนายน` | เฉพาะเดือน มิ.ย. (ปีปัจจุบัน) |
| `export เดือนกุมภาพันธ์ 2569` | ก.พ. ปี พ.ศ. 2569 (= ค.ศ. 2026) |
| `export คนมาสาย` | เฉพาะพนักงานที่มีวันมาสาย (ทั้งแถว) |
| `export 10 คนที่มาสายบ่อยที่สุด` | 🆕 Top 10 เรียงจากมาสายมากสุด (ได้ 10 คนพอดี) |
| `export เรียงจากเข้างานดีที่สุดไปแย่ที่สุด` | 🆕 ทุกคน เรียงดี→แย่ |
| `export ของ สมชาย เดือนมิถุนายน เฉพาะคนมาสาย` | รวมหลายเงื่อนไขพร้อมกัน |
| `export 2026-06-01 ถึง 2026-06-15` | ระบุช่วงวันที่ตรงๆ |

**คำที่ระบบเข้าใจ**
- 📅 เดือน: ชื่อไทยเต็ม/ย่อ (มกราคม–ธันวาคม, ม.ค.–ธ.ค.) และอังกฤษ (January/Jan…)
- 🗓️ ปี: พ.ศ. (แปลงเป็น ค.ศ. ให้) หรือ ค.ศ.
- 🏷️ สถานะ: *มาสาย, ออกก่อน, ขาดงาน/ไม่มา, ขาดสแกนเข้า, ขาดสแกนออก, ปกติ*
- 🔢 **จำนวน/อันดับ (Phase 10):** "10 คน / top 10 / อันดับ 10" (จำกัด Top N) · "ดีที่สุด→แย่ที่สุด", "แย่ที่สุด", "มาสาย/ขาด/ไม่สแกน **บ่อยที่สุด**" (จัดอันดับ)

> 💡 **กรองสถานะ:** ระบบจะเลือก "พนักงานที่มีสถานะนั้น" แล้วส่งออก **ทั้งแถว** ของเขา ไฟล์จึงยังเป็น Matrix ที่ **นำกลับมาอัปโหลดได้** (re-uploadable) ครบถ้วน
> AI จะตอบยืนยันเงื่อนไขที่จับได้ก่อน เช่น *"ส่งออกข้อมูล: ช่วง 2026-06-01 ถึง 2026-06-30 · เฉพาะสถานะ มาสาย"*

## 5. 🗄️ จัดการข้อมูล (Admin)
- **ดู/ค้นหา:** กรองตามชื่อ / ช่วงวันที่ / **สถานะ (7 แบบ)** → กด "ค้นหา"
- **Export Excel:** กดปุ่มส่งออก — ใช้ตัวกรองเดียวกับที่ตั้งไว้ ได้ไฟล์ Matrix re-uploadable
- **ลบข้อมูล (Danger Zone):** ลบตามช่วงวันที่ / ตามพนักงาน / เลือกเป็นรายการ / ล้างทั้งหมด — ทุกครั้งมีหน้าต่างยืนยันพร้อมจำนวนที่จะลบ

> ⚠️ การลบย้อนกลับไม่ได้ — แนะนำ Export สำรองก่อน

## 6. 👥 จัดการผู้ใช้ (Admin)
- **เพิ่ม:** "+ เพิ่มผู้ใช้" → email, password, ชื่อ, บทบาท, avatar
- **แก้ไข:** ไอคอนดินสอ → แก้ข้อมูล / เปลี่ยนรหัสผ่าน
- **ลบ:** ไอคอนถังขยะ → ยืนยัน
- **ค้นหา/กรอง:** ตามชื่อ/อีเมล/บทบาท
> Validation: email ถูกรูปแบบ, password ≥ 6 ตัว, ยืนยันรหัสผ่านต้องตรงกัน
